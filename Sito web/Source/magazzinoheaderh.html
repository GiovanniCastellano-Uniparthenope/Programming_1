Descrizione del file

Il file magazzinoheader.h presenta al suo interno le librerie utili al programma e i prototipi delle funzioni programmate


Oltre le librerie standard c e standard i/o, l'header include la libreria string.h utile per la funzione che richiede di inserire una stringa per il settore, e la libreria permette di usare le sue funzioni per eseguire proprio le operazioni su stringhe.
